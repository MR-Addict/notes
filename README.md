## 2. 简介

本项目基于 mdbook 框架，主要是用作笔记的功能，记录一些重要的内容，以方便我在不同的机器上使用 **Copy+Paste** 魔法。

因为是笔记的原因，所以我不会写得太详细，目的是达到我自己能看懂，日后有时间会逐渐完善更正相关内容。

- [https://mr-addict.github.io/notes](https://mr-addict.github.io/notes)
